# Análisis de tiendas. Proyecto Alura
## Introducción
Es el primer proyecto de tipo análisis financiero acerca de datos de ventas y costos de un caso ejemplo.
## Uso
El ejercicio tiene 3 secciones de análisis: facturación por tienda, promedio de clientes y costos de envío. Estas secciones
incluyen un gráfico de visualización que facilita la interpretación de datos.

Para poder usarlo es necesario clonar el repositorio. He instalar aquellos programas independientes.
